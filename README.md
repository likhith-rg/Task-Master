## Task Manager Application

This is a simple web-based application that allows users to manage their tasks efficiently. It implements basic CRUD (Create, Read, Update, Delete) functionality to manage to-do items.

### Features
- **Create**: Add new tasks to your to-do list.
- **Read**: View all your current tasks.
- **Update**: Edit existing tasks to keep them up-to-date.
- **Delete**: Remove completed or unwanted tasks.
